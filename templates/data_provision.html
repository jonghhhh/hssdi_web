데이터 제공 서비스

연구에 활용할 수 있는 다양한 사회과학 데이터를 수집하여 제공합니다

데이터 제공 개요

인문·사회과학 데이터 연구소(HSSDI)는 연구자들이 사회과학 연구에 활용할 수 있도록 다양한 형태의 데이터를 체계적으로 수집하고 제공하는 서비스를 운영하고 있습니다. 수집된 모든 데이터는 연구 목적으로 활용 가능하며, 개인정보 보호법 및 연구윤리 규정을 준수합니다.

제공 목적

  • 사회과학 연구의 데이터 기반 강화
  • 실증적 연구 방법론 확산
  • 연구 데이터의 재활용성 증대
  • 학제간 협력 연구 촉진

활용 대상

  • 인문·사회과학 분야 연구자
  • 대학원생 및 박사과정생
  • 정책 연구기관 관계자
  • 데이터 기반 연구에 관심 있는 모든 분

수집 데이터 유형

  • 메타데이터: 발행일시, 언론사, 카테고리
  • 감정 분석: 기사 톤 및 감정 점수
  • 키워드 태깅: 주요 이슈 및 키워드 추출

📊 데이터 활용 안내

활용 가능한 연구 분야

  • 미디어 연구: 뉴스 프레이밍, 의제 설정 분석
  • 정치 연구: 여론 변화, 정치적 극화 분석
  • 사회학 연구: 사회 갈등, 집단 행동 분석
  • 심리학 연구: 감정 분석, 인지 편향 연구
  • 경제학 연구: 시장 심리, 소비자 행동 분석
  • 언어학 연구: 언어 변화, 담화 분석

제공 데이터 형태

  • 원시 데이터: 수집된 원본 데이터
  • 전처리 데이터: 정제 및 구조화된 데이터
  • 분석 결과: 기본 분석이 완료된 데이터
  • 메타데이터: 데이터 수집 및 처리 정보
  • API 접근: 실시간 데이터 접근
  • 맞춤형 데이터셋: 연구 목적별 특화 데이터

데이터 보안 정책

  • 개인정보 비식별화 처리
  • 암호화된 전송 및 저장
  • 접근 권한 제한 및 감사
  • 데이터 보존 및 폐기 정책
  • 보안 취약점 정기 점검

연구윤리 준수

  • IRB(생명윤리위원회) 승인
  • 연구 목적 외 사용 금지
  • 데이터 재배포 제한
  • 연구 결과 공개 시 인용 필수
  • 정기적인 사용 현황 보고

데이터 신청 및 문의

신청 방법

  1. 데이터 신청서 다운로드 및 작성
  2. 연구계획서 및 IRB 승인서 첨부
  3. 이메일 또는 방문 신청
  4. 검토 및 승인 과정 (3-5일 소요)
  5. 데이터 사용 협약 체결

제공 비용

  • 기본 데이터셋: 무료
  • 전처리 데이터: 실비 (5-10만원)
  • 맞춤형 데이터: 별도 협의
  • * 학술 연구 목적은 우대
